This is a small console game for brain training. Here you can find five small games, below more about each game.

## Setup

```bash
make install
```

## Start to play games

### Brain-even game

Task: answer "yes" if the number is even, otherwise answer "no"

To run the game:
```bash
brain-even
```
or
```bash
make brain-even
```
